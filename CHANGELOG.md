# Changelog — Usage Attribution Reference

All notable changes to this reference are documented in this file.

The project follows a **versioned reference specification model**.
Changes reflect semantic clarification or boundary adjustments,
not feature or product development.

---

## v1.0.0 — Initial Reference Release  
**Date:** 2026-01

### Included
- Defined scope boundaries for usage attribution semantics
- Established a three-layer reference model:
  - Event
  - Usage
  - Attribution
- Documented semantic mappings from events to attribution
- Anchored definitions to stable, primary technical sources

### Notes
- This release intentionally excludes billing, settlement,
  pricing, payment, and commercial execution.
- The reference is designed for low change frequency.

---

## Versioning Policy

- Minor revisions clarify terminology or attribution boundaries.
- Major versions indicate structural or semantic changes.
- All changes are documented explicitly.
